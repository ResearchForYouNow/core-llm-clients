# Security Policy

## Reporting a Vulnerability
Please do not disclose security issues publicly or via regular GitHub issues.

Use one of the following private channels:
- GitHub Security Advisory (preferred): Open a draft security advisory in the repository's Security tab.
- Email: researchforyounow @gmail.com

When reporting, please include (as applicable):
- Affected versions and modules
- Type of issue and potential impact
- Steps to reproduce (PoC), sample inputs, logs
- Any known mitigations or workarounds
